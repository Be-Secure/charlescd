# CharlesCD's UI components documentation
<br/>
<br/>

## Overview

UI components are the building blocks to compose a user interface. This documentation shows CharlesCD's existing components.<br/>

## How to use

You can browse through the components in the sidebar, and click on `show code` to see its usage. You will be able to see the props of each component.

## Contribute

You are able to create a new component or extend existing ones, contribute with CharlesCD, check out the [**contribution guide**](https://github.com/ZupIT/charlescd/blob/main/CONTRIBUTING.md).<br/>
Note:  Before creating a component, please verify if there is one that already works for what you need.
